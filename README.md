**ReconZ** is a next-generation, high-performance reconnaissance dashboard designed for security researchers and bug bounty hunters. It streamlines the initial phase of infrastructure mapping by aggregating passive intelligence from multiple trusted sources into a stunning, glassmorphism-inspired interface.

> "Map your target's entire infrastructure footprint with zero contact — pure passive recon, no packets sent directly."

---

## 💎 Nebula Stealth Aesthetic
ReconZ isn't just a tool; it's an experience. Built with the **Nebula Stealth** design language, it features:
- **Glassmorphism UI**: Frosted glass effects, backdrop blurs, and layered depth.
- **Dark Mode First**: Optimized for long-night recon sessions with high-contrast neon accents.
- **Dynamic Animations**: Smooth transitions, counting statistics, and an interactive particle background.
- **Fully Responsive**: A seamless experience from ultra-wide monitors to mobile devices.

---

## 🚀 Key Features

### 🔍 Subdomain Reconnaissance
Aggregate certificate transparency logs and public DNS archives to discover hidden subdomains without touching the target's servers.
- **Intelligence Sources**: HackerTarget, URLScan.io, crt.sh, JLDC, CertSpotter, RapidDNS, and DNSRepo.
- **Real-time Status**: Live tracking of hits per source and unique IP resolution.
- **Export Capabilities**: Download results in clean text format (All subdomains or Resolved only).

### 🔗 Endpoint Discovery
Harvest API routes, hidden paths, and historical parameters from web archives.
- **Passive Harvesting**: Leverages Wayback Machine and Common Crawl.
- **Pattern Matching**: Extract sensitive paths and potential vulnerabilities from historical snapshots.

### 🔌 Post-Recon Workflow
Once your data is gathered, ReconZ provides a guided intelligence methodology for the next steps:
1. **httpx**: Live host detection & tech stack fingerprinting.
2. **naabu**: High-speed port scanning.
3. **ffuf**: Directory fuzzing for undocumented routes.
4. **nuclei**: Automated vulnerability scanning with community templates.

## 🛠️ Technical Stack
ReconZ is built to be lightweight, portable, and fast.

## 🚀 Quick Start
1. **Navigate to the Live URL**: Open the dashboard link provided above.
2. **Enter Target**: Input your target domain (e.g., `nasa.gov`) in the primary input field.
3. **Initiate Scout**: Click the button to start the multi-source passive discovery process.
4. **Analyze & Export**: Review the aggregated intelligence and download the results for your local toolkit.
